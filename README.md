## 설치 
[참고](https://mmistakes.github.io/minimal-mistakes/docs/quick-start-guide/)

### Ruby, Jekyll, bundler 설치하기
```bash
$ sudo apt install ruby ruby-dev build-essential
$ sudo gem install jekyll bundler
```

### gemfile 변경 사항 있는 경우
```bash
$ bundle install
```

## 실행
```bash
bundle exec jekyll serve
```
